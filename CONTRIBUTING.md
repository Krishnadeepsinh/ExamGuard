# Contributing

This hackathon repository is organized for daily, reviewable commits.

## Commit Style

Use concise conventional commits:

- `feat: add paper config validator`
- `fix: unify integrity thresholds`
- `docs: add DPDP checklist`
- `test: add tier3 orchestrator case`

## Local Checks

```bash
cd frontend
npm run build
```

Backend smoke check:

```bash
python scripts/seed_demo.py
```
