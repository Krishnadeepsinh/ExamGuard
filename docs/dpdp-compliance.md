# DPDP Compliance Checklist

- Explicit consent is shown before monitoring starts.
- Consent includes webcam gaze, microphone RMS level, answer analysis, and tab activity.
- Raw webcam frames never leave the student browser.
- Audio is not recorded; only RMS level events are sent.
- Students receive non-accusatory integrity summaries.
- Flagged grades are held for teacher review, not auto-punished.
- Appeal records are auditable.
- Reports and materials use private storage and signed URLs.
- Students may request data deletion in a production deployment.
